Install NodeJS:
-	http://nodejs.org/download/


Open a command prompt, navigate to the bootstrap folder and issue the following to grab the necessary dependencies (Less, JSHint, Recess, and UglifyJS):

npm install -g less jshint recess uglify-js



Run Make.bat
